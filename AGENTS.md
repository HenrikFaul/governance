# AGENTS.md

This repository is the canonical governance source for HenrikFaul repositories.

## Read order
1. `controller.md`
2. `agent_execution_rules.md`
3. `codingLessonsLearnt.md`
4. `business_process_automation_prompts.md`
5. local repo changelog and versioning artifacts in the consuming repository

## Operating rule
Treat clear user requests as execution instructions.
If Jira, GitHub, changelog, governance, or documentation updates are the natural next step, perform them without separate permission unless the action is destructive, external, production-affecting, or security-sensitive.

- Core controller: `controller.md`
