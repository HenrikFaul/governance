# ChatGPT Context Guide

Use these governance files as the primary context for ChatGPT-based development work:
- controller.md
- agent_execution_rules.md
- execution_authority_and_automation_rules.md
- codingLessonsLearnt.md
- ui_ux_rules.md
- business_process_automation_prompts.md

## Behavior expectation
Treat user requests as execution instructions when the next step is clear.
Do not ask repeated permission for GitHub, Jira, changelog, governance, or documentation updates when those are already implied by the request.

## Recommended reading order
1. controller.md
2. agent_execution_rules.md
3. codingLessonsLearnt.md
4. ui_ux_rules.md when UI is affected
5. latest changelog and versioning artifacts in the target repository
